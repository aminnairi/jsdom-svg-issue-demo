# jsdom-svg-issue-demo

## Requirements

- Git
- Docker
- Docker Compose
- GNU/Make

## Clone

```console
$ git clone https://github.com/aminnairi/jsdom-svg-issue-demo.git
$ cd jsdom-svg-issue-demo
```

## Installation

```console
$ make install
```

## Node demo

```console
$ make start
```

## HTML demo

```console
$ chromium index.html
```

Replace `chromium` with your favorite browser's startup command.
